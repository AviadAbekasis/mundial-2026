# 🏆 חיזוי מונדיאל 2026

כלי לניתוח וחיזוי תוצאות גביע העולם 2026 — מנוע Elo + Poisson (Dixon-Coles) + סימולציית Monte Carlo,
עם נתונים חיים מהטורניר.

## איך מריצים

לחיצה כפולה על **`עדכן וחזה.bat`** — מושך נתונים עדכניים, מריץ 50,000 סימולציות, ופותח את הדאשבורד.

או מהטרמינל:
```
python src/run.py            # 50,000 סימולציות (ברירת מחדל)
python src/run.py 20000      # מספר אחר
```

דרישות: Python 3.9+ ו-`requests` (`pip install -r requirements.txt`).

## מה הכלי נותן

- **סיכויי זכייה בגביע** לכל נבחרת
- **סיכויי העפלה** מכל בית + מרוץ המקומות השלישיים
- **כמה רחוק** כל נבחרת צפויה להגיע (1/8, 1/4, חצי, גמר, אלופה)
- **המשחקים הקרובים** — גולים צפויים (xG) והסתברות ניצחון/תיקו/הפסד לכל משחק

## איך זה עובד (4 שכבות)

1. **נתונים** — תוצאות חיות מ-ESPN + דירוגי כוח (Elo) מ-eloratings.net. הכל חינם, בלי מפתח.
2. **Elo → גולים צפויים** — פער הדירוג בין שתי נבחרות → מספר גולים צפוי (λ) לכל צד.
3. **מודל משחק** — Poisson + תיקון Dixon-Coles → הסתברות תוצאה ותוצאה מספרית.
4. **Monte Carlo** — 50,000 הרצות של כל הטורניר. משחקים שכבר שוחקו נכנסים כתוצאה אמיתית
   (לא מוגרלים); רק העתיד מוגרל. שובר-השוויון לפי חוקי FIFA 2026 (מפגש ישיר לפני הפרש שערים כללי).

## מבנה הקבצים

```
src/
  teams.py            מיפוי קודים/שמות/דגלים, נבחרות מארחות
  fetch_data.py       מושך ESPN + eloratings -> data/state.json
  bracket.py          מבנה הבראקט (R32..גמר), שובר-שוויון, שיבוץ מקומות שלישיים (Annex C)
  model.py            Elo -> λ -> Dixon-Coles, דגימת תוצאות + נוקאאוט (הארכה+פנדלים)
  simulate.py         מנוע Monte Carlo -> data/sim.json
  build_dashboard.py  בונה output/index.html (עברית, RTL, מותאם נייד)
  run.py              מריץ הכל ברצף
output/index.html     הדאשבורד
```

## מצב הפרויקט

- ✅ **שלב 1 — מנוע + דאשבורד (הושלם).** רץ מקומית, חינם, נתונים חיים.
- ⬜ **שלב 2 — שכבת אנליסט (Gemini):** הרכבים צפויים, פורמציה, טקטיקה, השעיות.
- ⬜ **שלב 3 — אוטומציה בענן:** GitHub Actions מריץ לבד + GitHub Pages לצפייה מהנייד.

## הערה על כיול

המודל מבוסס Elo בלבד (שלב 1) ונוטה להעדיף את הפייבוריטיות מעט יותר מבתי-ההימורים.
זה צפוי וניתן לכיול בשלב 3 (הוספת כושר/שונות). הכיוון נכון, המספרים סבירים.
